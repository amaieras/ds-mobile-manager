Garanție

Retururile sunt acceptate doar dacă sigiliile aplicate pe produse sunt intacte, iar produsele nu prezintă urme de deteriorare (mufe rupte, cablu flex, rupt/tăiat, geam fisurat, display spart, display pătat, etc. ). Pentru retur este necesară expedierea produsului defect la adresa noastră, iar acesta trebuie să fie ambalat corespunzător. Rambursarea sumei se face doar dacă produsul corespunde din punct de vedere optic / funcțional. Nu se acceptă produse ambalate, fără folii de protecție, fără sigilii.
